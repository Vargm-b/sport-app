# Estandares

## Frontend:

* HTML5
* CSS3
* JavaScript vanilla

## Backend:

* Node.js
* Express

## Base de datos:

* PostgreSQL en supabase

## Dependencias:

* express

## Versiones:

* Node.js 18 o superior
* npm 9 o superior
* Express 5.1.0
* pg 8.16.3
* dotenv 17.2.1


## Estructura base:

* proyecto-grupal/
  apps/
    brenda/
    	frontend
    	backend
    integrante4/
    	frontend
    	backend
    integrante5/
    	frontend
    	backend
  
## Estándares:

* variables en camelCase
* nombres de carpetas y archivos en minúscula y sin espacios
* uso de DB_USER, DB_HOST, DB_NAME, DB_PASSWORD, DB_PORT
* no subir node_modules ni .env
* no subir .env.example y package.json

## GitHub:

* un solo repositorio grupal
* rama principal main
* cada integrante trabaja en su rama feature/nombre

## Parte grupal:

* base de datos común
* conexión común
* repositorio común
* estándares y documentación

## Parte individual:
* cada integrante adapta su aplicación para funcionar con la base común
